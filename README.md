# SQL---Online-Book-Store
An interactive and user-friendly Online Book Store web application that allows users to browse, search, and purchase books seamlessly.

🚀 Project Overview
The Online Book Store Management System is designed to simplify bookstore operations by storing and managing all records digitally. It allows users to maintain book details, customer information, order history, stock availability, and payment records using SQL queries and relational database concepts.

✨ Features
📖 Manage book records (Add, Update, Delete, View), 
👤 Store customer details, 
🛒 Track book orders and purchases, 
📦 Manage stock/inventory, 
💳 Store payment information, 
🔍 Search books by title, author, or category, 
📊 Generate reports using SQL queries, 
🔗 Relational database with proper table connections

🛠️ Technologies Used
Database: PostgreSQL / SQL, 
Language: SQL Queries, 
Tool: Postgre SQL / PGADMIN4

🗃️ Database Tables
Books, 
Customers, 
Orders, 
Order_Details, 
Payments, 
Categories, 
Authors
